# 基金信号操作手册

**数据来源**：51 支基金历史净值回测（债基 13396 行 / 股基 26656 行有效数据）  
**收益指标**：持有 60 个交易日的累计净值中位收益 & 盈利率  
**更新时间**：2026-05-27

---

## 债基

> 适用：中银增利 163806 / 博时信用 050011/050111 / 德邦锐裕 010309 / 招商招悦 003157 / 国金惠盈 006760

**基准**：合理区间波动 → 60d 中位 +1.02% / 盈利率 74%

| 信号 | 连续天数 | 60d 中位 | 60d 盈利率 | 操作 |
|---|:---:|---:|---:|---|
| **良性上涨** | 第 1 天 | +0.78% | 64% | ✅ 买入 / 加仓 |
| | 第 5 天 | **+1.49%** | **70%** | ✅ 持续信号，可加仓 |
| **高估** | 任意 | +1.19% | 80% | 🟡 持有，不要卖 |
| | 第 10 天+ | **+1.17%** | **91%** | 🟡 越久越不要动 |
| **极端大涨 且 高估** | 任意 | +1.03% | 67% | 🟡 持有 |
| **极端大涨（未高估）** | 第 1 天 | +1.20% | 77% | 🟡 持有 |
| **上涨尾声** | 第 1 天 | +0.75% | 60% (20d: +1.70%/90%) | 🟡 短期还在涨，等 20 天再考虑 |
| **合理区间波动** | 任意 | +1.02% | 74% | 🟡 持有（基准） |
| **短期调整，估值中性** | 任意 | +0.81% | 69% | 🟡 持有，短期回调后大概率恢复 |
| **动力不强，但仍高估** | 任意 | +0.97% | 75% | 🟡 持有 |
| **极端大跌（未低估）** | 第 1 天 | +0.84% | 71% | 🟡 持有 |
| **动力不强，但低估** | 第 1–10 天 | +0.14% ~ +0.44% | 51–57% | ⚪ 不动 |
| | **第 15 天+** | **-0.20%** | **42%** | 🔴 考虑减仓，越久越差 |
| **极端大跌 且 低估** | 任意 | +0.26% | 50% | 🔴 不买，往往是继续下跌信号 |
| **短期调整，长期低估** | 任意 | -1.63% | 23% | 🔴 减仓 |

### 债基规律
- **动量 > 估值**：债基进入低估区通常是在下行途中，等动量（良性上涨）出现再入
- **「高估」持续越久越好**：债基在长期上行趋势中「高估」不代表顶部，继续持有
- **「短期调整，长期低估」是最强的负信号**：60d 只有 23% 盈利率

## 股基

> 适用：A 股行业主题 + QDII + 商品类等所有非债基金

**基准**：合理区间波动 → 60d 中位 +1.29% / 盈利率 57%

| 信号 | 连续天数 | 60d 中位 | 60d 盈利率 | 操作 |
|---|:---:|---:|---:|---|
| **极端大跌 且 低估** | **第 1 天** | **+8.61%** | **81%** | ✅✅ 立即买入，不要等 |
| **极端大涨（未高估）** | 第 1 天 | +1.37% | 58% | ✅ 可小仓买入 / 追涨 |
| **良性上涨** | 第 1 天 | +1.10% | 56% | ✅ 买入 / 持有 |
| | 第 10 天 | **+3.61%** | **63%** | ✅✅ 持续越久越强，持续加仓 |
| | 第 15 天 | **+5.14%** | **70%** | ✅✅ 强趋势，持有 |
| **动力不强，但低估** | 第 1 天 | +2.05% | 58% | ✅ 可小仓 DCA |
| | 第 10 天 | +2.14% | 62% | ✅ 持续低估可继续 DCA |
| **高估** | 任意 | +1.63% | 60% | 🟡 持有，不要卖 |
| **极端大跌（未低估）** | 第 1 天 | +2.22% | 61% | 🟡 持有 / 小仓买 |
| **合理区间波动** | 任意 | +1.29% | 57% | 🟡 持有（基准） |
| **短期调整，估值中性** | 任意 | +1.19% | 56% | 🟡 持有 |
| **上涨尾声** | 第 1 天 | +0.10% | 51% | ⚪ 基本是抛硬币，持有但不加 |
| **极端大涨 且 高估** | 第 1 天 | -0.54% | 46% | 🔴 小幅减仓，极端大涨后高估容易回调 |
| **短期调整，长期低估** | 任意 | -1.11% | 29% | 🔴 减仓，持续下行信号 |

### 股基规律
- **唯一的一级买点**：「极端大跌 且 低估」—— 55 次里 81% 盈利，中位 +8.61%，出现即买
- **「良性上涨」是真实趋势信号**：持续越久越强，QDII 和趋势品种尤其适用
- **「极端大涨 且 高估」是卖出信号**（与债基相反）：极端上涨叠加高估，60d 负收益
- **「高估」本身不是卖点**：盈利率 60% 仍正，信号本身只是"在高位"，趋势可能继续

## 两类通用对比

| 信号 | 债基 | 股基 |
|---|---|---|
| 「良性上涨」 | ✅ 买（+0.88%/65%） | ✅ 买（+1.63%/58%）|
| 「高估」 | 🟡 持有（+1.19%/80%） | 🟡 持有（+1.63%/60%）|
| 「极端大涨 且 高估」 | 🟡 持有（+1.03%/67%） | 🔴 减仓（-0.49%/48%）**相反！** |
| 「极端大跌 且 低估」 | 🔴 不买（+0.26%/50%） | ✅✅ 立即买（+8.61%/81%）**相反！** |
| 「短期调整，长期低估」 | 🔴 减仓（-1.63%/23%） | 🔴 减仓（-1.11%/29%）|
| 「动力不强，但低估」长期持续 | 🔴 减仓（第15天+ 变负） | ⚪ 可继续 DCA |

> **同一信号在债基和股基可能意义相反，切勿混用。**

## 局限性说明

- 「极端大涨 且 高估」股基 113 例，但结论仍受样本周期影响（主要 2018–2025）
- 「极端大跌 且 低估」股基 55 例中，2022 年纳指熊市期间有多次亏损（-11.5%），判断时需结合大环境
- 债基样本充足（>13000 行），结论更可靠；部分年轻股基（<750天有效数据）不纳入统计
